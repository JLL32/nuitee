Base Information

Base URL: http://localhost:4000/v1

Content Type: application/json

Rate Limiting: 2 requests per second, burst of 4

Hotel Endpoints

GET /v1/hotels

List hotels with filtering and pagination

Query Parameters:

search (string) - Search term for hotel names page (integer) - Page number (default: 1) page_size (integer) - Items per page (default: 20, max: 100) sort (string) - Sort field: hotel_id, name, city, country, rating, starts (prefix with - for desc)
200 Hotels retrieved successfully 400 Invalid parameters
GET /v1/hotels/{hotelID}

Get details for a specific hotel

Path Parameters:

hotelID (integer) - Unique hotel identifier
200 Hotel found 404 Hotel not found

Review Endpoints

GET /v1/hotels/{hotelID}/reviews

List reviews for a specific hotel

Path Parameters:

hotelID (integer) - Hotel identifier

Query Parameters:

search (string) - Search term for review content page (integer) - Page number page_size (integer) - Items per page sort (string) - Sort field: id, hotel_id, name, country, city, rating, starts (prefix with - for desc)
200 Reviews retrieved 404 Hotel not found
GET /v1/hotels/{hotelID}/reviews/{reviewID}

Get a specific review

Path Parameters:

hotelID (integer) - Hotel identifier reviewID (integer) - Review identifier
GET /v1/hotels/{hotelID}/reviews/{reviewID}/summary

Get AI-generated summary of a review

Path Parameters:

hotelID (integer) - Hotel identifier reviewID (integer) - Review identifier

Example Requests

List Hotels with Search

curl "http://localhost:4000/v1/hotels?search=grand&sort=-rating&page=1&page_size=10"

Get Hotel Details

curl "http://localhost:4000/v1/hotels/123"

Get Hotel Reviews

curl "http://localhost:4000/v1/hotels/123/reviews?page=1&page_size=5"
